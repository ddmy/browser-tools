### 浏览器常用工具集合
#### 安装 Installation
---
```
npm install c-browser-tools
```
####  使用方式 Usage
---
```
import Tools from 'c-browser-tools'
const { Cookie, Url, Image, localStorage } = Tools
```
#### 注意 Warning
> 个人研究学习使用，生产环境慎用。<br>
> For personal study and research, use cautiously in production environment！
